# 📖 About the Project

Smart Bharat AI is a next-generation **Generative AI-powered Civic Companion** developed for the **DEVENGERS PromptWars 2026 Hackathon**.

The platform acts as a digital assistant that helps Indian citizens navigate government services with ease. From finding government schemes and understanding documentation requirements to reporting civic issues and receiving multilingual AI support, Smart Bharat AI makes governance more accessible, transparent, and citizen-friendly.

# 🚀 Getting Started

Clone the repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

# 🔑 Environment Variables

Create a `.env` file and configure:

```env
VITE_FIREBASE_API_KEY=

VITE_FIREBASE_AUTH_DOMAIN=

VITE_FIREBASE_PROJECT_ID=

VITE_FIREBASE_STORAGE_BUCKET=

VITE_FIREBASE_APP_ID=

VITE_FIREBASE_MESSAGING_SENDER_ID=

VITE_GEMINI_API_KEY=
```
